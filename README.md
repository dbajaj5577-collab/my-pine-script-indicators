## 📈 Featured Scripts

### 1. XAU Hyper-Growth Engine - Elite Ed. (Optimized v5)
An advanced quantitative trading strategy specifically engineered for highly liquid and volatile assets like **Gold (XAU)** and **Bitcoin (BTC)**.

* **Dynamic Trend Filter:** Uses a Hull Moving Average (HMA) to instantly detect underlying market direction.
* **Volatility Expansion Filter:** Signal execution is gated by an Average True Range (ATR) filter, ensuring trades are only opened when market volatility is expanding.
* **Liquid Session Gating:** Built-in session filters restrict trading to peak liquidity hours (e.g., London/New York overlap).
* **Dynamic Risk Management:** Automatically calculates mathematical Stop Loss (SL) and Take Profit (TP) levels based on current market ATR, featuring an active Break-Even (BE) trigger.

## 🛠️ How to Use These Scripts

1. Open the `.pine` file of the script you want to use in this repository and copy the code.
2. Go to [TradingView](https://www.tradingview.com/) and open a chart.
3. Click on the **Pine Editor** tab at the bottom of the screen.
4. Paste the code into the editor, click **Save**, and then click **Add to Chart**.
5. Use the **Strategy Tester** tab to view historical backtest results and performance metrics.
